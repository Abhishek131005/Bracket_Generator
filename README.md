# Zemo Bracket Generator

Monorepo for the Zemo sports tournament platform.

## Apps

- `apps/api` - Node.js + Express + TypeScript backend
- `apps/web` - React + Vite + TypeScript frontend

## Quick Start

1. Install dependencies:

   npm install

2. Run backend + frontend together:

   npm run dev

3. Open app:

   http://localhost:5173

4. API base:

   http://localhost:4000/api
